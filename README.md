Powerful Messenger Bot by Rasel Mahmud

## Setup

1. Clone repo
2. Copy `.env.example` → `.env` এবং আপডেট করো
3. Run `npm install`
4. Run `npm start` or `npm run dev`

## Features

- VIP/Admin/General permission
- Modular commands
- Auto-react
- Auto-unsend on reaction
- Scheduler
- Logging
- Database support
- Multi-language (i18n)
- Plugins support
